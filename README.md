# AI Second Brain

An AI-powered second brain system for managing knowledge, projects, and tasks.

## Structure

- `apps/` — Application services (web, api, worker)
- `docs/` — Project documentation (PRD, architecture, roadmap)
- `prompts/` — System and task prompts for AI agents
- `.claude/` — Claude Code skills and configuration

## Getting Started

```bash
docker-compose up
```
